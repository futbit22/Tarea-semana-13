```
METODO calcularSalario(horasT, pago)
    total ← horasT * pago
    RETORNAR total
FIN METODO

ALGORITMO Principal
    IMPRIMIR "Calculadora de salario por semana"
    LEER horasT
    LEER pago
    salarioTotal ← LLAMAR calcularSalario(horasT, pago)
    IMPRIMIR "El pago total es: " + salarioTotal
FIN ALGORITMO
```
